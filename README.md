## Getting Started

1. Clone the repo:

   ```bash
   git clone https://github.com/iva-stasia/metaverse-website.git
   ```

2. Install NPM packages:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```
